About JP2Forge Web

A web interface for the JP2Forge JPEG2000 conversion library, providing an easy-to-use system for converting and managing image files in the JPEG2000 format.

Features

Interactive Dashboard

Comprehensive dashboard with conversion statistics, storage metrics, and job monitoring.

Image Conversion

Convert images to JPEG2000 format with various compression options.

Multiple Document Types

Support for photograph, heritage document, color, and grayscale document types.

Parallel Processing

Process multiple conversion jobs simultaneously with Celery task queue.

User Authentication

User accounts with personal job management and history.

Detailed Reports

Conversion reports with quality metrics like PSNR and SSIM.

JPEG2000 Conversion

The application provides a user-friendly interface for JP2Forge's powerful conversion capabilities:

Compression Modes

Mode Description
lossless No data loss, larger file size
lossy Higher compression with data loss
supervised Quality-controlled compression with analysis
bnf_compliant BnF standards with fixed compression ratios

Document Type Options

Type Description
photograph Standard photographic images (default)
heritage_document Historical documents with high-quality settings
color General color images
grayscale Grayscale images

Quality Analysis

  • PSNR (Peak Signal-to-Noise Ratio) calculation
  • SSIM (Structural Similarity Index) analysis
  • Compression ratio reporting

Job Management

Real-time Progress

Track conversion progress in real-time with automatic status updates.

File Downloads

Download converted JPEG2000 files individually or in batch.

Job Cleanup

Manage and delete conversion jobs when no longer needed.

Retry Failed Jobs

Easily retry failed conversions with a single click.

Dependencies

  • Django 4.2
    Web framework
  • JP2Forge 0.9.1
    JPEG2000 conversion library
  • Celery 5.3.1
    Distributed task queue
  • Redis
    Message broker for Celery
  • ExifTool
    For metadata handling
  • PostgreSQL
    Database (for production)
